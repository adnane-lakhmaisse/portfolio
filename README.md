## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
/app
  /components
    /sections    # Page sections (Hero, About, Projects, etc.)
    /ui          # Reusable UI components
  /lib           # Utility functions
  /styles        # Global styles
/page.tsx        # Main landing page
/layout.tsx      # Root layout
```

## Customization

- Update project data in `components/sections/projects.tsx`
- Modify experience entries in `components/sections/experience.tsx`
- Adjust skills in `components/sections/skills.tsx`
- Update social links in `components/sections/contact.tsx`
- Replace placeholder images with your own

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Components:** shadcn/ui
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Smooth Scroll:** Lenis
- **Type Safety:** TypeScript
